🚀 HOW TO START
───────────────────────────────────────────────────────────────────────────────

STEP 1: SETUP BACKEND (5 minutes)
───────────────────────────────────────────────────────────────────────────────

  cd C:\Users\lauda\Desktop\SWV
  
  # Create environment
  python -m venv venv
  venv\Scripts\activate
  
  # Install dependencies
  pip install -r requirements.txt
  
  # Start server

🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Q: "ModuleNotFoundError: No module named 'backend'"
A: Make sure you're in the project root: C:\Users\lauda\Desktop\SWV
   Then run: python -m uvicorn backend.main:app --reload

Q: "Port 8000 already in use"
A: Use different port: python -m uvicorn backend.main:app --port 8001

Q: "YOLO model download fails"
A: Manually download: python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"

Q: "Slow analysis"
A: Use smaller YOLO model (yolov8n is fastest)
   Or run on GPU if available

Q: API returns 401 Unauthorized
A: Check API_KEY in request header matches settings
